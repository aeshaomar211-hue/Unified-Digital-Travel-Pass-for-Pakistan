 SmartSafar
 Overview

SmartSafar is an intelligent travel and transportation management application designed to simplify route discovery, booking, and travel planning. The system provides users with an easy way to search routes, book seats, and manage their travel history, while administrators can manage routes, buses, and bookings efficiently.

 Features
 User Features
User registration and login
Search available routes between cities
Book tickets / seats
View booking history
Manage profile
 Admin Features
Add / update / delete routes
Manage buses and schedules
View all user bookings
Manage users
